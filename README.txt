Simple ERP application based on REST and REACTIVE

A (very) simple ERP application with microservices which communicate with each 
other by way of REST or Reactive programming. The frontend is a testpage for sending
an order an follow the changes in inventory and shipping.

Techstack:

Frontend
- REACT, TS, MUI
- RESTful API between frontend and backend

REST
- Java 19
- Spring Boot, RestTemplate

REACTIVE
- Java 19
- Jakarta 10
- REACT, TS, MUI
